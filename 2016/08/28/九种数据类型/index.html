九种数据类型

发表于 2016-08-28

整型

byte

  • 位数: 8位
  • 表示的范围: -2^7 至 2^7-1（即-128至127，因为在java中0是正数，所以正数最大值是127，而负数的最小值是-128）
  • 封装类: Byte
  • 默认值: 0

short

  • 位数:16位
  • 表示的范围:-2^15 至 2^15-1(-32768 - 32768 )
  • 封装类: Short
  • 默认值: 0

int

  • 位数:32
  • 表示的范围:-2^31 至 2^31-1(-2147483648-2147483648 )
  • 封装类: Integer
  • 默认值: 0

long

  • 位数:64
  • 表示的范围:-2^63 至 2^63-1(-9233372036854477808-9233372036854477808 )
  • 封装类: Long
  • 默认值: 0L

浮点型

float

  • 位数: 32
  • 表示范围:1.4×10^ -45 至 3.4×10^38
  • 封装类: Float
  • 默认值: 0.0L

double

  • 位数:64
  • 表示范围:4.9×10^-324至1.8×10^308
  • 封装类：Double
  • 默认值: 0.0d

布尔型

boolean

  • 位数:1
  • 表示范围: true false
  • 封装类：Boolean
  • 默认值: false

字符型

char

  • 位数:16
  • 表示范围:0 至 2^16-1 (前128位和ASCII兼容)
  • 封装类：Character
  • 默认值: ‘\u0000’
